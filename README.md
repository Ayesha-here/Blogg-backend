# MERN Blog Application

A full-stack MERN blog application where users can register, log in, create blogs, upload images, and manage their own posts.

## Features

- User Authentication using JWT
- Secure Login & Registration
- Password Hashing with bcrypt
- Create and Delete Blogs
- Upload Blog Images using Multer
- Protected Routes
- MongoDB Database Integration
- REST API Architecture

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
- bcryptjs

### Frontend
- React.js
- Axios
- React Router DOM
